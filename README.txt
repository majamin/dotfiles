=== KDE dotfiles ===

Some basic dotfiles that I use for KDE.

Usage
=====

This is the KDE branch.

Pick a branch for the best results.

`master` branch .............. barebones
`KDE` branch ................. master + settings for KDE
`wayland` branch ............. master + settings for wayland WMs

Key files
=========

  ~/.profile ................. System environment variables, $PATH, etc.
  ~/.zshenv .................. Shell environment
  ~/.config/zsh/.zshrc ....... Shell config
  ~/.local/bin/ .............. Helpful scripts, etc.
